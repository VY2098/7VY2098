Nand to Tetris

Description:

The Nand to Tetris project is an engaging journey through the construction and functioning of a general-purpose computer system, starting from the ground up. It begins with the simplest logical gate, the NAND gate, and culminates in the realization of a fully functional computer system capable of running Tetris.


Achievements:
  • Successfully built a functioning computer system from Nand gates.
  • Designed and implemented a hardware platform and software hierarchy, revealing the interplay between hardware and software.
  • Gained practical experience with hardware description languages (HDLs) and simulated the designed hardware.
  • Developed an assembler to translate programs written in assembly language into machine code.
Technologies Used:
  • Hardware Description Language (HDL) for designing the computer architecture.
  • A Hardware Simulator to test and confirm the functionality of the designed hardware components.
  • An Assembler developed in a high-level language to translate assembly code into machine code.
  • The Hack platform, which includes a CPU, memory, and a screen, to run the final Tetris game.

Multi-thread Server

Description:

This project involves the development of a multi-threaded server capable of handling file transfers concurrently across multiple threads. The server employs robust thread synchronization mechanisms, using semaphores and mutex locks to manage access to shared resources. The primary functionality includes not only the transfer of files but also the computation and ranking of the frequency of specified keywords within all transmitted documents, providing a comprehensive analysis of the data traffic.


Achievements:
  • Implemented a multi-threaded server that efficiently handles concurrent file transfers on a local network.
  • Engineered a keyword frequency analysis tool that computes the occurrence rates of given terms across all transferred files.
  • Utilized semaphores and mutex locks to ensure thread-safe operations and prevent data races during file transfer and analysis.
  • Devised an algorithm for ranking the processed data, offering valuable insights into keyword significance and document relevancy.
Technologies Used:
  • Multi-threading in C and Python to enable parallel file transfers and data processing.
  • Semaphore and mutex lock mechanisms for maintaining data integrity and synchronization among threads.
  • A custom-built keyword frequency analyzer to parse and compute term frequencies from the file corpus.
  • An efficient sorting algorithm to order keyword occurrences, facilitating quick retrieval and data analysis.

Digits Clusters

Description:

The Digits Clusters project is a foray into the field of unsupervised machine learning, specifically using the K-means clustering algorithm to categorize handwritten digits. Starting from scratch, the project aims to identify and group numerous digit images by their similarities, effectively learning to recognize the numerical value each image represents without pre-labeled data.


Achievements:
  • Developed an unsupervised learning model to classify handwritten digits into distinct clusters.
  • Applied the K-means algorithm to analyze patterns and features in the data, facilitating the grouping of similar digits.
  • Processed and prepared a large dataset of handwritten digit images for input into the machine learning model.
  • Refined the model parameters to optimize cluster accuracy and minimize within-cluster variances.
Technologies Used:
  • K-means clustering algorithm for identifying inherent groupings in the dataset.
  • Data preprocessing techniques to convert images into a suitable format for analysis.
  • Dimensionality reduction methods, such as PCA, to enhance the clustering performance and computational efficiency.
  • Visualization tools to interpret and validate the clustering results, ensuring the model's effectiveness.
